# Simple-Vehicle-Price-Prediction-System

A simple full-stack, data-driven web application designed to estimate the market value of used vehicles. By leveraging a custom machine learning model, this system provides price predictions based on key vehicle attributes such as Brand, Year of Manufacture and Mileage.

## Overview

This project bridges the gap between data science and web development by integrating a robust backend predictive model with a clean, interactive user interface. The underlying model is specifically engineered to handle high-cardinality categorical data, seamlessly processing over 1,500 distinct models across 50 different vehicle brands to deliver accurate estimates.

## Dataset

The predictive model was trained and evaluated using the car_price_dataset.csv, which was sourced from Kaggle.

## Technologies Used

Machine Learning & Data Processing : Python, Jupyter Notebook (model.ipynb) and Pickle for model serialization (predictor.pickle).
Backend Web Framework : Python (app.py).
Frontend UI : HTML5 (index.html) and CSS3 (style.css).
